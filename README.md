# c51-calculator

适用于stc89c52RC芯片的51单片机的计算器程序，使用矩阵按键和独立按键输入，然后通过解析表达式得出计算结果
### 按键:

|  7   |  8   |  9   |  +   |
| :--: | :--: | :--: | :--: |
|  4   |  5   |  6   |  -   |
|  1   |  2   |  3   |  x   |
|  0   |  .   |  /   |  =   |

|  (   |  )   | 退格 | shift |
| :--: | :--: | :--: | :---: |
|  <   |  >   | 清屏 | shift |
